Video Streaming Platform (High Scale):

Components:
- AWS ALB
- Kong API Gateway
- OAuth Auth Service
- Streaming Service
- Metadata Service
- Recommendation Service
- Redis Cluster
- Cassandra Storage
- PostgreSQL Metadata DB
- Cloudflare CDN
- Kafka Event Queue
- Prometheus + Grafana

Architecture:
Clients access content via CDN which serves cached media globally. Requests for metadata pass through API Gateway and are handled by microservices. Kafka streams user activity events to the recommendation system. Cassandra stores high-volume streaming data while PostgreSQL manages structured metadata.

Scaling:
CDN handles global traffic distribution. Cassandra scales horizontally for high write throughput. Redis cluster caches metadata for fast access. Kafka partitions events for parallel processing across services.

Constraints:
- Enterprise scale
- Global users
- High throughput (>1000 RPS)
