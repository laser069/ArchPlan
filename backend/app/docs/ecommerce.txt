E-commerce Platform:

Components:
- AWS ALB
- API Gateway
- Auth Service
- Product Service
- Cart Service
- Order Service
- Payment Service
- Redis Cache
- PostgreSQL DB
- Elasticsearch Search
- S3 Storage
- Monitoring Service

Architecture:
Users browse products via Product Service. Cart Service manages user carts in Redis. Order Service processes checkout and interacts with Payment Service. Elasticsearch enables fast product search.

Scaling:
Redis caches product listings. PostgreSQL uses read replicas. Search is offloaded to Elasticsearch. Services scale independently.

Constraints:
- Moderate scale
- 100–500 RPS
